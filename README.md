# Домашнее задание к работе 11
## Условие задачи 
Найти среднее значение положительных элементов массива, расположенных по нечетным индексам. Массив состоит из N вводимых с клавиатуры положительных и отрицательных чисел.
## 1. Алгоритм и блок-схема
### Алгоритм
1. Объявить массив A[N]
2. Объявить переменные:
   + 'sum' - Сумма всех положительных элементов массива расположенных по нечетным индексам
   + 'del' - Колличество положительных элементов массива расположенных по нечетным индексам
3. Ввести значения элементов массива с клавиатуры:
   + Если индекс элемента нечетный и сам элемент больше 0:
     + Прибавить значение элемента к переменной 'sum'
     + Прибавить 1 к переменной 'del'
4. Выводим среднее значение, то есть переменную 'sum' делим на переменную 'del'

## 2. Реализация программы 
``` C
#define _CRT_SECURE_NO_DEPRECATE
#include <stdio.h>
#include <stdlib.h>
#define _USE_MATH_DEFINES
#include <math.h>
#define N 10

task();



void main()

{

	system("chcp 1251");

	task();

}


task()
{
	//я 19 варинт
	int A[N];
	float sum = 0;
	float del = 0;

	printf("Программа для подсчета среднего значения положительных элементов массива, состоящего из %i целых чисел, располженных по нечетным индексам\n", N);
	for (int i = 0; i < N; i++)
	{
		printf("A[%i] = ", i);
		scanf("%i", &A[i]);
		if (i % 2 != 0 && A[i] > 0)
		{
			sum += A[i];
			del++;
		}
	}
	printf("Итого среднее значение равно >>> %.2f", sum / del);


}
```
